#### git 常用操作
```
// git pull 相当于
git fetch origin <branch>
git merge FETCH_HEAD

// 合并分支
git fetch origin <branch>
git merge <branch>

// 放弃本地修改，使用最新的远程代码
git fetch origin <branch>
git reset --hard FETCH_HEAD

// 创建并切换到分支
git checkout -b <branch>
```
